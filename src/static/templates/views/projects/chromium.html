I am a contributor to the chromium and blink projects. I am just starting out but hope to become a regular contributor.

Patches

These are the patches I've got into Chromium and Blink so far:-

  1. Don't hide cursor when cmd key pressed on mac. - When using keyboard shortcuts, e.g. cmd+T, chromium would hide the pointer unnecessarily. This was particularly obvious in the case of opening a tab - issue.
  2. Use proportional line height for menu items to prevent clipping. - Setting the font size to 'Very Large' in settings caused the items of the lefthand menu in this page to clip - issue.
  3. Dispatch DOM FocusIn/Out events when the page changes focus. - Prerequisite for 5.
  4. Make content shell activate and focus dev tools on show. - Prerequisite for 5.
  5. Don't dispatch blur/focus events if the element's page is not focused. issue - Unfortunately this was later reverted due to a test outside of blink flaking.
  6. Fix closing tags, missing quotes in test HTML file. - Trivial cleanup - issue.
  7. Ensure view focused on first tab in FocusTraversal test. - Fixes flakiness for commit 5/8 - issue.
  8. (Ongoing - not yet committed) - Redux of 5 - issue.
